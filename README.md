SLIDER-WR
=========


Slider-wr es un plugin wordpress, gestion simple de slider.

	<?php
	  /*
	  * Slider Wr Simple
	  *
	  * @param string, int $arg (wp_item.ID, wp.title).
	  * @return string.
	  *
	  **/
	  echo slider_wr($arg);
	?>
	
Utiliza librerias js 

	jquery.js
	jquery.bxslider
	elita.validate.js
